# SeoKiller

Klonowalne repozytorium Next.js do budowania portali SEO z automatycznie generowaną treścią.

## Instalacja

```bash
pnpm install
cp .env.example .env
```

Dodaj swój klucz API Gemini do `.env`:
```
GEMINI_API_KEY=your_key_here
```

## Użycie

```bash
pnpm dev
```

## Konfiguracja

Cała konfiguracja portalu znajduje się w pliku `site.config.json`.

## Deploy

Projekt gotowy do wdrożenia na Vercel.
